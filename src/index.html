Aerotech Programming Course

The course where you'll learn the basics of collaborative development from the ground up

Why this course?

It is a known fact that tailored education works better when it comes to reduced groups of people that might or might not have prior experience in the topic. This course will allow both beginner and experienced users to enjoy the learning process, discover new functionalities and even getting new ideas for their own development.

How to follow the course?

It is really to follow this course, since each content is extensively explained and organized in a way that the learning curve is smooth and beginner-friendly, without sacrificing complex information. You will start by setting up your environment for collaborative development, then you will learn the basics of Python programming in order to be able to develop your own projects and collaborate with other people.

Environment setup

Since this course starts from scratch, an environment setup is required. This includes download, installation and configuration of a version control system (Git), a programming language (Python), a code editor (Visual Studio Code) and external tools that make collaborative development easier for beginners and experienced users alike (such as GitHub Desktop).

Python programming

Due to the fact that this programming course focuses on a beginner-friendly learning methodology, Python was chosen as programming language. It is a high-level programming language, which means that it is easier to read and write than other languages such as C++ or Java. It is also a multi-paradigm (general-purpose) language, which means that it can be used for almost anything, from web development to data science or machine learning.
