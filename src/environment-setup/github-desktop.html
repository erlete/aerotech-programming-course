Introduction

GitHub Desktop is the application interface for GitHub Web. It allows a more direct interaction between the user and GitHub/Git features that are available on the web.

Why is GitHub Desktop important?

I personally recommend its usage, specially for beginners, since it makes working with the interface an easy task. Using Git directly might be the cause for plenty of difficulties, yet advanced users are more than welcome to use it. Furthermore, the application is well-designed, which means that it does not sacrifice much customization or functionality for the sake of simplicity.
