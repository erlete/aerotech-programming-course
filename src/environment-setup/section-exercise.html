Section Exercise

Complete the Environment Setup section by deploying a local server in a local copy of the course repository

Introduction

In this exercise, you will use all the technologies you have installed through the Environment Setup section to deploy a local server in a local copy of the course repository. This will allow you to see the course content in your browser and interact with it in an editable mode, which is mandatory for the remaining exercises and project.

Objective

This section exercise aims to ensure that you have successfully installed all the required technologies and that you are able to interact with them in a minimal way. If you are able to complete this exercise, you will be granted access to the remaining sections of the course.
How to solve the exercise?

Use the side navigation buttons to move between steps. Each step contains a set of instructions that you need to follow to solve the exercise.
