Introduction

Python is a programming language that lets you work quickly and integrate systems more effectively. It is a high-level, interpreted, interactive and object-oriented scripting language. Python is designed to be highly readable. It uses English keywords frequently whereas other languages use punctuation, and it has fewer syntactical constructions than other languages.

Why is Python important?

Python lets you do heavy work in just a few, easy-to-write lines of code. Since it is multi-paradigm (although OOP-dominant), you can use whichever programming style you are used to (e.g. functional). It is not the fastest of languages out there, but for sure the one that gives you more results for less effort.

Managing Python versions

Since the language is constantly evolving, there are several versions of Python available, with the most recent being the 3.13.0. This constant upgrade troubles many Python users due to a very simple problem: they do not know how to properly manage Python versions. This is why in this course, you will use a tool named pyenv (a simple Python version manager).
